About Sigon

Sigon is an agent description language, created to be a simple and powerfull way to implement different cognitive architectures as multi-context systems. Its intrpreter is Java-based, and includes a Eclipse plugin to compile and test your own multiagent systems.

The language was created by the Artificial Intelligence and Educational Technology Laboratory from the Federal University of Santa Catarina, Brasil. It's objective is operationalize agents as multi-context systems, allowing more flexibility in the agent creation process and in the communication betwen agent's components.
